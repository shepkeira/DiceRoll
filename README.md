# DiceRoll

## Set Up
1. Install Python 3.9.4

## Running the Program
1. ```python GUI.py```

## About this program
This is a dice roller and card drawing program.
The dice roller allows you to roll dice from a 6 dice system (Dice with sides 4, 6, 8, 10, 12, 20). You can roll as many of the same die you want at once. You can also add numbers to your roll such as you would need in a TTRPG like D&D.
In the card drawing program you are able to draw cards into your hand. You can draw as many cards as you want into your hand at once.
